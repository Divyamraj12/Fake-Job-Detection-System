# 🚀 Fake Job Detection System

An AI-powered web application that detects whether a job posting is **real or fake** using Machine Learning and Natural Language Processing (NLP).

---

##  Project Overview

Many fake job postings scam users by offering high salaries with no experience.  
This system helps users identify fraudulent job listings by analyzing job descriptions.

---

## 🔧 Tech Stack

-  Python  
-  Scikit-learn (Machine Learning)  
-  Flask (Web Framework)  
-  HTML, CSS, Bootstrap  

---

##  Features

-  Detect fake vs real job postings  
- NLP-based text analysis (TF-IDF)  
-  Keyword-based fraud detection  
-  Confidence score prediction  
-  Modern multi-page UI (Flask)  
-  Contact page with social links
